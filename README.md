# GraphAI

Understand any codebase visually.

GraphAI analyzes a repository and generates an interactive graph of modules, files, and dependencies to help developers understand large codebases instantly.

---

## Demo

Paste a repository URL and instantly explore its architecture.

Example workflow:

repo → analysis → interactive graph

Example architecture output:

Frontend → API → Services → Database

---

## Features

- Automatic repository analysis
- Interactive dependency graph
- Module clustering
- Entry point detection
- Architecture visualization
- Multi-language support
- Simple CLI interface
